# Ecommerce-SQL-Analysis-3

## Project Description

This project shows how to analyze ecommerce sales data using SQL.
I used queries to filter, sort, group, and analyze data to find useful business insights.

## Tools Used
*  MY SQL Workbench 8.0 CE
* CSV Dataset

## Skills Demonstrated

* SELECT, WHERE, ORDER BY
* GROUP BY
* SUM and AVG functions
* JOINS
* Subqueries
* Views
* Indexes

## Files Included

* Dataset (CSV file)
* SQL queries file
* Output screenshots

## How to Run

1. Create database
2. Import dataset
3. Run SQL queries
